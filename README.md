## A Multi-Tenant E-Commerce Platform for Farmers

This is a platform I made to make the selling process easier for small farmers. Users can enter the products they have for sale to their virtual "farm stands" and have their customers place their orders on the platform. Payment is integrated with Stripe. 

## Learning

This project taught me how to use many full-stack development tools and technologies, like NoSQL databases, TypeScript, React, and Next.js. It also taught me how to think about building large platforms, by getting into the weeds of code while also keeping the big picture in mind. 

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
